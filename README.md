# Student Report Card Generator

A Python program that takes student scores,
calculates the average, and prints a full report
card with grades for each subject.

## What it does
- Stores scores for multiple subjects
- Calculates letter grades (A/B/C/D/F)
- Calculates average and final grade
- Prints a formatted report card

## How to run it
python report_card.py

## Built with
- Python 3
